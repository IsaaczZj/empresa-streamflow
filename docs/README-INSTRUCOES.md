# Instruções da Atividade

Este repositório faz parte da atividade prática da disciplina de **Documentação Técnica em DevOps**.

## Objetivo do grupo
Documentar um runbook operacional da StreamFlow para padronizar atuação técnica em incidentes e reduzir tempo de indisponibilidade.

## Documentação que deve ser produzida
- Arquivo final: `docs/runbook.md`
- Referência de estrutura: `docs/runbook-template.md`

## Regras da simulação
- Esta atividade é uma simulação acadêmica.
- Não é necessário alterar infraestrutura nem implementar automações reais.
- O foco é documentação operacional e tomada de decisão durante incidentes.

## Forma de trabalho
- O trabalho deve ser realizado no fork do grupo.
- O grupo deve deixar explícitos gatilhos, critérios e responsabilidades.
